# Исходные данные

Поместите в этот каталог исходные файлы, которые не хранятся в Git:

```text
200_magn_stab.txt
200_lla_stab.csv
201_magn_ori.txt
201_lla_ori.csv
```

Для запуска одного набора данных достаточно соответствующей пары файлов:

- `analyze_200_stab.m`: `200_magn_stab.txt` и `200_lla_stab.csv`;
- `analyze_201_ori.m`: `201_magn_ori.txt` и `201_lla_ori.csv`.

Файлы магнитометра и исходные навигационные CSV исключены из репозитория правилами `.gitignore`.
