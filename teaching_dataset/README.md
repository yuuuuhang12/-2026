# Учебный полусинтетический набор спутниковых данных

Этот набор данных предназначен только для учебных задач по анализу спутниковых измерений.

## Методика

1. Используются реальные навигационные точки спутника Polytech-Universe.
2. Для каждой точки рассчитывается теоретическое магнитное поле по модели IGRF-14.
3. К теоретическому модулю магнитного поля добавляются остатки, оцененные по реальным измерениям.
4. Для учебных целей добавляются контролируемые аномалии.

## Основные файлы

- `data/teaching_satellite_dataset.csv` — основной набор данных.
- `data/anomaly_description.csv` — описание внесенных аномалий.
- `data/metadata.json` — параметры генерации и единицы измерения.
- `figures/` — контрольные графики.

## Основные поля

- `timestamp_utc` — время точки в UTC.
- `altitude_km`, `latitude_deg`, `longitude_deg` — положение спутника.
- `B_model_nT`, `B_model_mGs` — теоретический модуль поля по IGRF-14.
- `residual_mGs` — остаток, оцененный по реальным данным.
- `B_teaching_mGs` — учебное значение после добавления остатка и аномалий.
- `is_anomaly`, `anomaly_type`, `anomaly_id` — разметка учебных аномалий.

## Ограничение

Данные являются полусинтетическими и не должны использоваться как научные наблюдения.
