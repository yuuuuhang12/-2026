# Polytech-Universe MATLAB Analysis

Локальный Git-репозиторий с материалами обработки данных летней практики.

## Содержимое

- `scripts/` — MATLAB-скрипты обработки данных:
  - `analyze_200_stab.m`
  - `analyze_201_ori.m`
- `results/figures/` — PNG-графики для отчета.
- `results/tables/` — CSV-таблицы со статистикой обработки.
- `results/summaries/` — текстовые сводки анализа.
- `docs/` — инструкции по запуску.
- `WORK_LOG.md` — журнал выполненной работы.

## Что не входит в репозиторий

Исходные данные из папки `01 Задание` не включены в Git. В частности, не сохраняются большие файлы магнитометра `*_magn_*.txt`, исходные навигационные таблицы `*_lla_*.csv`, документы `.docx` и MATLAB-файлы `.fig`.

Причина: исходные магнитометрические файлы имеют размер около 70 MB каждый и не подходят для хранения в обычном Git-репозитории.

## Запуск

Открыть MATLAB в папке `scripts/` и выполнить:

```matlab
analyze_200_stab
analyze_201_ori
```

Скрипты ожидают, что исходные данные находятся в исходной структуре проекта:

```text
02 Work/01 Задание/
```

## Основные результаты

- Для набора `200_stab` построены графики магнитного поля, наземного трека и таблицы статистики.
- Для набора `201_ori` сохранены исходные результаты и дополнительная версия `201_ori_filtered`, где удалены одиночные экстремальные выбросы `|B| > 750 мГс`.
